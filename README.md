# 剑三小铺

web入门所写的第一个小应用，网游虚拟物品商店。

本项目使用Node + Express + Materialize + MongoDB开发。

## 功能

* 使用不同组件花式展示商品

## 主要使用

前端:
* jquery
* materialize

后端:
* node
* express
* mongoose
* mongodb


## 构建

1. 安装依赖

  ```bash
  npm install
  ```
2. 运行

  ```bash
  node bin/www  
  # server at http://127.0.0.1:3000
  ```
